Student Projects Showcase

Overview

Welcome to our Student Projects page! Here you'll find an impressive collection of projects created by Computer Science students at Berea College. These projects represent both individual and group efforts, showcasing the creativity and technical skills of our students. Browse through different projects using the navigation bar on the left.

Submit Your Project

Would you like to showcase your project? We encourage students to share their work with the community! To submit your project, please email the lead TA with the following information:

  • Project title
  • Brief description of your project
  • The motivation behind creating this project
  • Course information (if the project was created for a class)
  • Internship information (if the project was created for an Internship)
  • LinkedIn profile (optional)
  • GitHub repository link (optional)

Project Categories

Our showcase includes projects from various domains of computer science, including:

  • Web Development
  • Mobile Applications
  • Data Science and Analytics
  • Artificial Intelligence and Machine Learning
